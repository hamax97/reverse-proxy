## PROJECT REQUIREMENTS

1. Install ``` virtualenv ``` and ``` pip ``` packages
2. Run the commands
   ``` $ virtualenv environment -p /usr/bin/python3 ```
   ``` $ source environment/bin/activate ```
   ``` $ pip install django ```
   ``` $ environment/webserver1/runserver.sh ```
   ``` $ environment/webserver2/runserver.sh ```
